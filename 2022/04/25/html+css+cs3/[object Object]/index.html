4.标签的分类*

（1)单双标签

单标签：只有开始标签，又叫空元素即内部不可嵌套其他内容（img br input hr）

双标签：有开始标签，有结束标签，关闭符不可以省略，内部可嵌套其他标签内容

（2)块，行块，行内标签

块：独占一行，宽高生效，内外边距皆可控，宽度默认继承父级100%，内部可嵌套任何标签 ul table form div nav header p h1-h6

行块：按内容占大小，宽高生效，可以与行内，行块同一行展示，内部可以嵌套文本或其他行内标签 img input video

行内：按内容占大小，宽高不生效 span a i strong

(3)语义化和非语义化

语义化：p h img a hr 居多

无意义：div span

（四）路径!!!

分类

路径分为相对路径和绝对路径

1.绝对路径（较少使用）

以本地磁盘盘符开头如：c：\user:\img.png

以http开头如：htttp://www.4399.com

使用方便，不可更换位置，比如换不同的电脑显示失败

2.相对路径

从当前文件开始去找到目标文件的过程，需要去查找相对路径

优点：文件相对位置不变的情况下，在不同的电脑上，不同的盘符都能正常显示

（十三）浮动*

1.语法

样式名：float: 取值：none（默认） lrft（左浮动） right（右浮动）

2.特点

1.脱离标准文档流，块/行块/行内都可加float，加了浮动的标签会变成一个行块标签
2.用法：外层用标准文档流，内层浮动盒子，一般内层浮动的盒子不会超出父级，兄弟级之间最好保持一致，实现同一行展示

3.清除浮动

方法一：给父级加固定的高 优点：简单易行 缺点：内容数量不定时会溢出 使用场景：内容数量确定

方法二：加空的

div>,加样式，clear{clear：both}

样式名clear 表示清除 取值 left 清除左浮动 取值 right 清除右浮动 both：清除两边浮动 none不清除（默认）常用取值both

空div加在浮动结束的位置

优点：实用性好 缺点：找到结束浮动的位置 ，页面代码冗余（空div多，可以在父级加空伪类处理）

适用场景：所有需要清除浮动的地方

方法三：父级无定高，父级中加overflow：hidden

优点：简单易行 缺点：子元素有定位时会失效

适用场景：无定位时可用

2.rotateX（ndeg）沿着x轴旋转n度

3.rotateY（ndeg）沿着y轴旋转n度

n是数字，可正可负，deg是单位

3.缩放

scale（x，y）对宽度缩放x倍，对高度缩放y倍

scaleX（x）对宽度缩放x倍

scaleY（y）对高度缩放y倍

缩小：0<倍数<1 缩小 -1<倍数<0 翻转缩小

放大：倍数>1放大 倍数<-1翻转放大

临界值:1原本大小 0没有/隐藏 -1原本大小翻转

4.倾斜/拉伸

skew（ndeg）沿着x轴和y轴同时倾斜/拉伸n度

skewX（ndeg）沿着x轴倾斜/拉伸x倍

skewY（ndeg）沿着y轴倾斜/拉伸y倍

n是数字，可正可负，正数表示拉左上和右下角在水平或垂直方向拉伸

负数表示拉右上和左下角在水平或垂直方向拉伸 deg是单位

（七）transform-origin基点

1.样式名

transform-origin

2.取值

①px单位的长度值

②%

③关键词 水平方向：left right center 垂直方向：top center bottom

z轴只能用①

3.作用

在2d和3d转换效果中，确定不动的点在哪里，默认值都是中心点（矩形对角线的交点）

（八）3d效果必须在父级添加的样式

1.transform-style：preserve-d：

转换效果按照3d效果展示——搭建3d舞台

2.perspective：1000px；

元素离视图的距离、透视——视角

（九）渐变

定义：背景在两种或多种颜色之间平滑过渡

由浏览器生成，可以写background：渐变或background-image：渐变，本质是图片

分类：线性渐变，径向渐变

1.线性渐变

background：linear-gradient（颜色1，颜色2，。。。）默认从上至下

background：linear-gradient（to left方向,颜色1，颜色2，。。。）指定最后一个颜色的方向

background：repeating-linear-gradient（颜色1 %，颜色2 %，。。。）重复的线性渐变%后一个比前一个大

2.径向渐变

background：radial-gradient（颜radial色1，颜色2，。。。）默认均匀分布

background：radial-gradient（颜radial色1 %，颜色2 %，。。。）不均匀分布

background：radial-gradient（circle，颜radial色1，颜色2，。。。）默认椭圆，改为圆

background：repeating-radial-gradient（颜色1 %，颜色2 %，。。。）重复的径向渐变%后一个比前一个大

（十）过渡transition

1.定义

同一个样式名，样式值从一种逐渐变成另一种

2.简写、复合样式

transition：样式名 花费时间 时间曲线 等待时间

无顺序，浏览器会以看到的第一个时间为花费时间 样式名 花费时间必写。

3.单独样式

1.transition-property

默认值none，表示无过渡，all表示所有样式名都可过渡

某个样式名 width等，表示只有这个样式有过度效果

2.transition-duration花费时间 正数

默认值是0，表示无过渡

取值可以是s或者是ms 1s=1000ms

3.transition-delay 等待时间

默认值是0，表示无等待

取值可以是s或者是ms 1s=1000ms

为正表示等待多久开始过渡 为负表示提前多久开始过渡

4.transition-timing-function 时间曲线

默认ease 先慢后快

linear匀速 ease-in 慢速开始 ease-out慢速结束 ease-in-out 慢速开始和结束

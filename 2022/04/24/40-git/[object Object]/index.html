配置

局部配置用户名

git config user.name wuhengjun

全局配置用户名

git config –global user.name wuhengjun

局部配置用户邮箱

git config user.email 2043563166@qq.com

全局配置用户邮箱

git config –global user.email 2043563166@qq.com

环境配置

Windows 平台

//文件模式

git config –global core.filemode false

//自动校准

git config –global core.autocrlf false

macOS 系统

git config –global core.autocrlf false

改变了文件或文件夹名称的大小写，默认情况下 git 是无法检测到的

关闭大小写忽略

git config –global core.ignorecase false

查看配置

当前配置

git config –list

Repository配置(储存库)

git config –local –list

全局配置

git config –global –list

系统配置

git config –system –list

当前目录下隐藏资源
ls -lA

初始化

初始化一个空的Git仓库

git init

添加忽略文件

touch .gitignore

常用指令

指令过多清屏

clear

状态

untracked 未跟踪

committed 已提交

modified 被修改

staged 已暂存

新建文件=>工作区

工作区=>暂存区 git add filename/.

工作区=>本地仓库 git commit -am “注释”

暂存区=>本地仓库 git commit -m ‘注释’

暂存区=>工作区 git rm –cached filename

本地仓库=>暂存区 git reset –soft HEAD

本地仓库=>工作区 git reset –mixed HEAD

本地仓库=>全部消失 git reset –hard HEAD

快捷方式HEAD^ HEAD为当前多一个^为前一个

查看状态

git status

查看目录

ls -a

查看日志

git log

q退出

简要日志

git log – oneline

最近的num次日志

git log – oneline -num

具体到时间的展示

git log –graph –date=relative –pretty=tformat:’%Cred%h%Creset -%C(auto)%d%Creset %s %Cgreen(%an %ad)%Creset’

查看文件内容

cat filename

进入编辑状态编辑某个文件

vim filename

退出按Esc 输入三个字符 :wq回车

清除修改

放弃掉所有还没有加入到缓存区的修改

但不会删除掉刚新建的文件

单文件

git checkout – filename

所有文件

git checkout – .

最近一次的提交的信息进行修改

git commit –amend

插入字符串到文件

echo “string” >> filename

忽略

忽略后缀为txt的文件

*.text

除外

！a.txt

忽略根目录下的name文件

/name

忽略name下的所有文件

name/

忽略name下的后缀为text的

name/*.text

忽略name目录下的所有后缀text的

name/**/*.text

分支

查看

本地

git branch

远程

git branch -r

本地和远程

git branch -a

创建分支

git branch name

删除分支

git branch -d name

切换分支

git checkout name

创建并切换到新建分支

git checkout name

当前分支与指定分支合并

git merge name

默认加–ff 快进

git merge name –no-ff 推荐

git merge name –squash 适用commit多时(合并后要commit一次)

变基

git rebase

当其他开发者开发功能提交在你前面时，

就是把你的提交点更改为master最新提交点

git rebase master

再切换到master

git checkout master

合并

git merge name –no-ff

///////////

查看已经合并到当前分支

git branch –merged

查看没有合并到当前的分支

git branch –no-merged

查看各个分支最后一个提交对象的信息

git branch -v

删除远程分支

git push origin -d name

重命名分支

git branch -m oldname newname

拉取

与远程仓库关联

git remote add origin url

查看

git remote -v

拉取

git pull origin master

上传(第一次加 -u)

git push -u origin master
